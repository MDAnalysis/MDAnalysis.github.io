We have just released MDAnalysis version 0.16.1. This is a pure bug fix release.
Highlights are listed below but for more details see the [release notes][notes].

In this release we have 4 new contributors Jon Kapla, Sang Young Noh, Andrew
William King and Kathleen Clark. Besides the bug fixes we updated the style
([PR 1126][doc-pr]) of our [docs][docs] so that they match with the website.

# Upgrade

You can upgrade with `pip install --upgrade MDAnalysis` . If you use the conda
package manager run `conda update -c conda-forge mdanalysis`
